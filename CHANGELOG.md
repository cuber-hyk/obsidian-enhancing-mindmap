# 变更日志

本项目所有值得记录的变更都会写入本文件。

格式遵循 Keep a Changelog；适用时采用语义化版本。

## [Unreleased]

### Added

- 增加 MindMaster 风格节点框选和多选整组迁移，支持 `Ctrl`/`Meta` 追加选择以及整组撤销和重做。
- 增加链接标题显示设置，可选择在链接图标旁直接显示链接标题。
- 增加脑图右下角导航控件，提供小视图、视口定位、缩放滑动条、隐藏/恢复按钮和 hover 四角拖拽缩放点。
- 增加导航控件节点数统计，显示当前可见节点数与脑图总节点数。

### Fixed

- 修复多选后拖动画布会被后续空白点击事件清空选择的问题，只有静止空白单击才取消选择。
- 修复按住 `Ctrl`/`Meta` 框选节点时滚轮误触发缩放的问题，框选期间可滚动画布并累计跨屏选择。
- 修复脑图缩放后导航控件小视图视口框位置偏移的问题。

## [0.2.6] - 2026-07-05

### Added

- 在脑图节点编辑状态中增加可视化工具栏，可插入外部链接、Vault 文件以及 Vault 或本地图片。
- 外部链接和 Vault 文件链接图标增加悬停标题、右键编辑及删除操作，链接变更支持撤销和重做。
- 图片附件在编辑态支持点击选中、拖拽缩放，并可通过 `Backspace` 或 `Delete` 删除。

### Changed

- 脑图节点内的外部链接和 Vault 链接改为仅显示链接图标，编辑态也隐藏原始 Markdown 链接地址，减少长路径、链接标题和蓝色下划线对节点阅读与布局的干扰。
- 节点新增和删除统一为 MindMaster 风格键盘操作：`Space` 进入编辑，`Backspace` 删除选中的非根节点，`Enter` 保存编辑或新增同级节点，根节点使用 `Enter` 新增一级子节点，`Shift+Enter` 在节点内换行，`Tab` 新增下一级节点。
- 删除节点后优先选中同级节点，没有同级节点时才回退到父节点。
- 链接标题与节点正文分离；新增、编辑或删除链接不会替换节点正文。
- 新插入图片使用默认节点宽度，编辑态显示图片本体而不是图片 Markdown 文本。
- 节点中的 Vault 文件和外部链接始终在新标签页打开，不再替换当前脑图视图。
- 最低 Obsidian 版本提升至 1.5.7，以使用官方附件路径与 Markdown 链接 API。

### Deprecated

### Removed

- 移除节点右侧旧新增按钮，以及 `Alt+Shift+Enter`、`Shift+Insert` 节点新增命令。

### Fixed

- 修复节点新增链接后图标短暂显示又消失的问题，保存编辑内容时只执行一次阅读态渲染。
- 修复带链接节点在布局刷新后链接图标空间丢失的问题，分支与子节点会按链接图标预留宽度重新布局。
- 修复编辑态输入内容后按 `Enter` 保存可能失去节点选中态的问题。
